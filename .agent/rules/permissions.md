System & Debug Permissions
debugmode.activate: Grants the ability to toggle the system-wide Debug Mode via the Konami Code.

Risk Level: High (Exposes real-time system events and raw data).

Default Assignment: System-Administrators only.

1. Debug Mode Access: The debugmode.activate permission is a 'Reserved' permission. It should only be granted to 'System-Admin' type users and should never be used by standard feature plugins.
